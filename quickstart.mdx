## Getting Started with Vidoc

Welcome to Vidoc! Vidoc is a security automation platform that allows you to automate your Recon and Security Testing. Follow this guide to get started with Vidoc.

## Create a Vidoc Account

To get started with Vidoc, you need to create an account. You can create an account by visiting [https://app.vidocsecurity.com](https://app.vidocsecurity.com).
